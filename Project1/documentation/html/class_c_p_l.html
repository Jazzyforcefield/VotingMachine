CPL class.

This class is CPL. It is used for create a entity of CPL class inherited from VotingType. An entity of CPL class can have attributes like parties, remainder, remainder to index have methods like increment, get_parties, CalculateWinners, CalculateWinners GenerateRandomNumber, BreakTie, create_txt_file.
